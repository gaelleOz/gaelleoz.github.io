Comment venir?

NOTRE FILE D'ATTENTE VIRTUELLE

Nous avons mis en place un système de liste d'attente virtuelle avec l'app SKEEPIT

Le concept :

Vous venez au restaurant directement sans réservations. Premier arrivé, premier installé. Plus vous venez tôt (on ouvre à 10h), plus vous avez des chances d'avoir une table directement ...

Lorsque le restaurant est plein, nous ouvrons une file d'attente virtuelle via l'application SKEEPIT, (la file n'est pas ouverte si le restaurant n'est pas plein) Il vous suffit alors de vous inscrire dessus. Le logiciel vous enverra directement un sms quand la table est prête pour vous. Ainsi vous pouvez vous balader, faire les courses, profiter de la vue pendant l'attente, pas besoin de rester devant le restaurant. Scan ce QR code ou Télécharge l'app de SKEEPIT et rejoins la file d'attente virtuelle lorsque le restaurant est plein!
