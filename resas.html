NOTRE FILE D'ATTENTE VIRTUELLE

Scanne ce QR code pour t'inscrire sur la file d'attente virtuelle!

Nous ne prenons pas de réservations mais nous avons mis en place un système de liste d'attente virtuelle avec l'app SKEEPIT!

Vous venez au restaurant directement sans réservation. Premier arrivé, premier installé. Plus vous venez tôt (on ouvre à 10h), plus vous avez des chances d'avoir une table directement!

Comment ça marche?

  1. 1 Lorsque le restaurant est plein, nous ouvrons une file d'attente virtuelle via l'application SKEEPIT, (la file n'est pas ouverte si le restaurant n'est pas plein).

  2. 2 Il vous suffit alors de vous inscrire dessus.

  3. 3 Le logiciel vous enverra directement un SMS quand la table est prête pour vous.

  4. 4 Ainsi vous pouvez vous balader, faire les courses, profiter de la vue pendant l'attente, pas besoin de rester devant le restaurant.
