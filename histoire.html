ETYMOLOGIE

Le déjeuner (de.ʒœ.ne) : n.m singulier (fin XIIe siècle). Littéralement,« arrêter le jeûne », du latin disjejunare, puis disjunare « briser le jeûne ».

"DESJEUNEUR"

Le desjeuneur est donc celui qui arrête le jeûne.
Devenez vous aussi un desjeuneur en brisant le jeûne avec de délicieux plats!

CONCEPT

Le desjeuneur vous propose une carte de petits-déjeuners, de plats du jour, pâtisseries, soupes, croques...
Ici...vous pouvez bruncher TOUS les jours !
